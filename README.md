#ZombieWall

一个面向末日生存整合包的轻量 Forge 模组：让僵尸在靠近玩家时更像“围堵”，而不是全挤在同一侧推着玩家走。

#功能

近距离围堵分流：僵尸进入玩家一定范围后，会自动选择玩家四侧（前/后/左/右）中僵尸更少的一侧移动，补齐缺口，形成更自然的包围。

减少“僵尸堵车”：重定位过程中临时禁用僵尸之间的推挤，避免被前排僵尸卡住。

反发呆：玩家静止不动时，部分僵尸可能不主动重算路径；本模组会在必要时强制刷新追击，减少站桩发呆。

#版本

Minecraft 1.20.1

Forge 47.x（在 47.4.10 环境开发/测试）

#使用

把 *.jar 放进整合包的 mods/ 文件夹即可。无需配置文件（如需参数化可自行改代码/发 PR）。
